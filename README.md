# 구름(Goorm)

한국어 형태소 분석을 통한 워드클라우드 생성기 구름.

## 설치

```bash
git clone https://github.com/nyanye/Goorm.git
sh INSTALL
```

## 사용법

```bash
python3 goorm.py cat.txt cat.png
```

![cat](/examples/cat.png)

```bash
python3 goorm.py food.txt apple.png
```

![food](/examples/apple.png)
